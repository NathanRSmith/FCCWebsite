HTML Files and Structure

About RENO template

RENO is Multipurpose HTML5 template suitable for companies or individuals that want to present their work at the highest level. It is based on Twitter Bootstrap framework. This template uses the latest HTML5 and CSS3 technologies.
To change colors just change color-1.css file in "header tag" for the color you would like to use (color-2.css,... up to color-8.css).

Columns

Template is divided into rows and columns, with respective classes .row and .col-md-* (col-lg-*, col-sm-*, col-xs-*), where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12). Look at the examples below for better explanation and some real examples.

Full-width: To achieve a full-width column, use the code below:

Container

There's one more special class which is being used across all HTML files, which is .container. This class determines the maximum width of a page and also makes sure that page is center-aligned. It also determines rendering of a template on mobile devices and tablets. This class can have also different backgrounds.
